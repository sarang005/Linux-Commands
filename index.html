Communication Commands

dig

Query DNS server

Syntax

dig [options] domain [query-type]

Options

Option Description Example
@server Use specific DNS server dig @8.8.8.8 google.com
MX Mail server records dig gmail.com MX
+short Short output dig google.com +short
-x Reverse lookup dig -x 8.8.8.8

Examples

Basic DNS Lookup
dig google.com
Find Mail Servers
dig gmail.com MX

ssh

Securely connect to remote systems

Syntax

ssh [options] user@host

Options

Option Description Example
-p Specify port ssh -p 2222 user@server
-i Identity file ssh -i key.pem user@server
-X Enable X11 forwarding ssh -X user@server

Examples

Connect to Remote Server
ssh user@192.168.1.10
Run Command Remotely
ssh user@server "df -h"

File Comparison Commands

diff

Compare files line by line

Syntax

diff [options] file1 file2

Options

Option Description Example
-u Unified format diff -u old.c new.c
-w Ignore whitespace diff -w file1 file2
-r Recursive compare diff -r dir1 dir2

File Management Commands

ls

List files and directories

Syntax

ls [options] [directory]

Options

Option Description Example
-l Long listing ls -l
-a Show hidden files ls -a
-h Human readable sizes ls -lh

Examples

Basic Listing
ls
Detailed Listing
ls -lah

chmod

Change file permissions

Syntax

chmod [options] mode file

Permission Values

4 = Read
2 = Write
1 = Execute

Examples

Full Permission
chmod 777 script.sh
Recursive Permission
chmod -R 755 project/

rm

Remove files and directories

Examples

Delete File
rm file.txt
Delete Directory
rm -rf project/

Warning

Be careful with rm -rf. It permanently deletes files and directories.

Media Commands

genisoimage

Generate ISO image from a directory

Syntax

genisoimage [options] source_directory

Examples

Create ISO
genisoimage -o backup.iso /data
